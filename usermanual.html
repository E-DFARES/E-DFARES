Manual E-DFARES


Step for Admin

1. Register Admin Account

  • Fill in the following details:
    • IC Number
    • Name
    • Phone Number
    • Address
    • Email
    • Password

2. Login to Your Account

  • Use your email and password to log in.

Inside Admin Page

  • Register Users and Technicians
  • Admin can register new user and technician accounts.
  • Add Product
  • Admin can add a product by entering its serial number.
  • View Reports
  • Admin can view reports made by users.
  • Approve Orders
  • Admin needs to approve orders, which will then be sent to technicians for processing.

Step for User

1. Get Login Information from Admin

  • Login using the provided admin ID, email, and password.

Inside User Page

  • View Added Products
  • User can view products added by admins.
  • Make Report
  • User can create a report about a product by typing the problem in the report form.
  • Make Order
  • User can place an order for a product. The order will be sent to the admin for approval or rejection.
  • Contact Admin
  • User can click the "Contact Admin" button to view the admin's contact info, including name, phone number, and address.

Step for Technician

1. Get Login Information from Admin

  • Login using the provided admin ID, email, and password.

Inside Technician Page

  • View Reports
  • Technician can see reports made by users.
  • Send Feedback
  • Technician can send feedback about the reports.
  • View Approved Orders
  • Technician can view orders approved by the admin from users.
