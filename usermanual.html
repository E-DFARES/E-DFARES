Admin Page User Manual

Overview

The Admin Page allows administrators to manage profiles, products, reports, orders, and feedback, as well as to register new users and technicians. The interface is designed to be user-friendly, with a sidebar for easy navigation and a main content area for detailed views and interactions.

Header and Navigation

  • Admin Name: The name of the logged-in admin is displayed dynamically.
  • Sign Out Button: Click this button to sign out of the admin portal.

Sidebar Menu

The sidebar on the left provides quick access to various sections of the admin portal:

  • Profile: View and manage your profile information.
  • Product: View and manage the product list.
  • Report: View and manage reports.
  • Order: View and manage orders.
  • Chat: View and manage feedback (chat).
  • Contact Us: View contact information.
  • User Manual: View the user manual.
  • Register User: Register a new user.
  • Register Technician: Register a new technician.
  • Sign Out: Sign out of the admin portal.

Profile Management

  • Admin ID: Displayed at the top. Use the "Copy Admin ID" button to copy it to the clipboard.
  • Admin Name, IC, and Email: Displayed below the admin ID.

Product Management

  • Product List Table: Displays product details including name, serial number, and date.
  • Delete Button: Click the trash can icon to delete a product.

Report Management

  • Report List Table: Displays report details including user name, product, serial number, report, and date.

Order Management

  • Order List Table: Displays order details including user name, product, and status.
  • Approve/Reject Buttons: Click these buttons to approve or reject orders.

Feedback Management

  • Chat Container: Displays the chat history.
  • Chat Input: Type your message and click "Send" to reply to user feedback.

Contact Us

In the Contact Us section, you can view contact information.

User Manual

In the User Manual section, you can view this user manual for reference.

Register User

Click the Register User button to register a new user. This will prompt you to enter the necessary details for user registration.

Register Technician

Click the Register Technician button to register a new technician. This will prompt you to enter the necessary details for technician registration.

Sign Out

Click the Sign Out button in the header or sidebar to sign out of the admin portal.

Technician Page User Manual

Introduction

Welcome to the E-DFARES Technician Page. This manual will guide you through the various features and functionalities available to you.

Profile

The Profile section displays your personal information. To view your profile, click the "Profile" button in the sidebar. Your details such as name, IC number, and email will be displayed.

Report List

In the Report List section, you can view all the reports submitted. To access this section, click the "Report" button in the sidebar. A table will show the user name, product, serial number, report details, and date.

Order List

The Order List section allows you to see all approved orders. Click the "Order" button in the sidebar to view a table listing the user full name, product, and approval status.

Feedback

The Feedback section is where you can communicate via chat. Click the "Chat" button in the sidebar to access the chat interface. You can type your message in the input field and click "Send" to submit it.

Contact Us

For any assistance or inquiries, navigate to the "Contact Us" section by clicking the respective button in the sidebar. Here you will find customer support details, contact information, business hours, and social media links.

User Manual

This section contains the user manual itself. You are currently viewing it.

Sign Out

To sign out of your account, click the "Sign Out" button located in the top navigation bar.

Navigation Tips

Use the sidebar to navigate between different sections. Your last visited section will be remembered and opened automatically the next time you log in.

User Page User Manual

Welcome to the User Page

This page is designed to provide you with all the tools and information you need to manage your products, reports, orders, and more. Below are the details for each section available on this page:

Profile

In the Profile section, you can view your personal information including your full name, IC number, and email address.

Product List

The Product List section allows you to add new products by entering the product name and serial number. You can view and manage your products in a table format.

Send Report

In the Send Report section, you can select a product, enter its serial number, and write a report. This report will be submitted for review.

Order

Use the Order section to place orders for products. Simply enter the product name and submit your order.

Chat

The Chat section provides a platform to communicate with support. You can type your message and send it directly to the support team.

Contact Us

In the Contact Us section, you can find the necessary information to get in touch with the support team for any queries or assistance.

User Manual

This section contains the user manual which provides an overview of the different sections and how to use them effectively.

Buttons and Actions

Each section can be accessed using the sidebar buttons. The system will remember your last clicked button and automatically take you to that section upon your next visit.

If you have any questions or need further assistance, please use the Contact Us section to get in touch with our support team.
